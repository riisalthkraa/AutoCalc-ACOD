# async calculateProfitabilityKPIs

Calcule les KPIs de rentabilité à partir des factures et matériaux. Analyse les factures payées du mois en cours et calcule marges, taux de marque et couverture données.

Workflow :

  1. Charge matériaux via IPC 'get-materials' (pour purchaseCost)
  2. Filtre factures payées du mois en cours
  3. Pour chaque article des factures :
    • Récupère purchaseCost depuis material
    • Calcule marge : priceHT - (purchaseCost × quantity)
    • Cumule totalMargin, totalPriceHT, totalPurchaseCost
  4. Calcule KPIs finaux :
    • Marge moyenne % = (totalMargin / totalPriceHT) × 100
    • Taux de marque = totalPriceHT / totalPurchaseCost
    • Couverture = (articles avec coût / total articles) × 100

View Source modules/revenue/RevenueModule.js, line 877

Example
const kpis = await calculateProfitabilityKPIs(allInvoices);
// → { avgMarginPercent: 28.5, totalMargin: 3450.00, dataCoverage: 85.2, markupRate: 2.15, ... }

# async loadRevenueData

Charge et affiche toutes les données CA et rentabilité. Méthode principale qui orchestre tous les calculs et affichages.

Workflow :

  1. Charge toutes les factures via IPC 'list-invoices'
  2. Calcule CA mensuel (calculateMonthlyRevenue)
  3. Met à jour KPIs CA (updateRevenueKPIs)
  4. Crée graphique Chart.js (createMonthlyRevenueChart)
  5. Affiche tableau historique (displayRevenueHistory)
  6. Calcule KPIs rentabilité (calculateProfitabilityKPIs)
  7. Met à jour KPIs rentabilité (updateProfitabilityKPIs)

Déclenchée par :

  • Clic sur onglet #revenue-tab (attachEventListeners)
  • Appel manuel depuis d'autres modules

# updateProfitabilityKPIs

Mettre à jour l'interface avec les KPI de rentabilité

Met à jour les 4 cartes d'indicateurs dans l'onglet "CA du Mois" :

  • Marge Moyenne (% avec code couleur : vert >30%, orange >15%, rouge <15%)
  • Marge Totale (en euros)
  • Couverture Données (% articles avec coût d'achat)
  • Taux de Marque (coefficient multiplicateur)
