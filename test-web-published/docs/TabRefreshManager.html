TabRefreshManager

TabRefreshManager()

TAB REFRESH MANAGER

Système centralisé de rafraîchissement automatique des onglets. Chaque module peut enregistrer une fonction de refresh qui sera appelée automatiquement lorsque l'utilisateur ouvre l'onglet correspondant.

Architecture:

  • Detection automatique des événements Bootstrap tab (shown.bs.tab)
  • Throttling intelligent (évite refresh si < 2s depuis dernier refresh)
  • Support du cache avec TTL configurable par onglet
  • Force refresh optionnel (bypass cache)

// Refresh simple avec cache 30s

  await revenueModule.loadRevenue();
}, { alwaysRefresh: true });
