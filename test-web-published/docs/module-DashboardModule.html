DashboardModule

======================================== DASHBOARD MODULE - Tableaux de bord et analytics

Module de gestion des tableaux de bord pour l'application AutoCalc OptiDevis. Fournit deux dashboards distincts selon le mode utilisateur (Admin/Vendeur) avec KPIs, graphiques Chart.js et analytics avancés.

Architecture : Le module gère deux dashboards distincts avec des fonctionnalités adaptées au rôle :

  • Dashboard Vendeur : Vue simplifiée avec KPIs essentiels et top 10 articles
  • Dashboard Admin : Vue complète avec analytics avancés et graphiques détaillés

========================================== DASHBOARD VENDEUR (Onglet "Accueil")

Fonctionnalités :

  • KPI 1 : Devis en attente (avec pastille rouge animée si > 0)
  • KPI 2 : Devis créés cette semaine
  • KPI 3 : Recherches matériaux effectuées aujourd'hui
  • Top 10 : Articles les plus recherchés cette semaine (avec bouton sélection rapide)

Données utilisées :

  • Devis (list-saved-quotes) pour KPIs 1 et 2
  • Historique de recherche localStorage pour KPIs 3 et Top 10

========================================== DASHBOARD ADMIN (Onglet "Tableau de bord")

KPIs Admin (4 indicateurs) :

  1. CA du mois : Chiffre d'affaires du mois en cours (factures payées uniquement)
  2. Devis en attente : Nombre de devis au statut "en_attente" (avec badge rouge)
  3. Factures en retard : Factures échues ou dont la dueDate est dépassée (avec badge rouge)
  4. Taux de conversion : % devis acceptés/facturés sur total devis

Graphiques Chart.js (4 graphiques) :

  1. Répartition des devis (Doughnut) : Distribution par statut avec pourcentages
  2. Évolution du CA (Line) : CA sur les 6 derniers mois (factures payées)
  3. Top 20 Produits (Bar horizontal) : Produits les plus vendus du mois par CA
  4. Performance Catégories (Doughnut) : Répartition du CA par type de matériau

Analytics avancés :

  • Tendances produits : Comparaison mois actuel vs mois précédent
  • Détection automatique : Produits en hausse (>+5%), en baisse (<-5%), nouveaux, disparus
  • Recommandations intelligentes : Suggestions d'actions selon les tendances
  • Seuils de variation : Hausse forte (>50%), moyenne (>20%), faible (>5%)

========================================== TRACKING & ANALYTICS

Historique de recherche :

  • Enregistre chaque recherche de matériau (id, nom, type, prix, timestamp)
  • Stockage localStorage (clé: 'materialSearchHistory')
  • Utilisation : KPI "Recherches aujourd'hui" + Top 10 articles

Analyse des données :

  • Agrégation par produit et catégorie
  • Calcul des quantités vendues et CA par produit
  • Comparaison temporelle (mois actuel vs précédent)
  • Génération automatique de tendances et recommandations

========================================== GESTION DES GRAPHIQUES CHART.JS

Lifecycle des graphiques :

  1. Destruction des anciens graphiques (évite les fuites mémoire)
  2. Création avec nouvelle configuration
  3. Stockage dans this.adminDashboardCharts pour accès ultérieur

Plugins personnalisés :

  • percentagePlugin : Affiche % sur graphiques doughnut (si >= 3%)
  • revenueValuesPlugin : Affiche valeurs € sur graphique ligne CA
  • topProductsValuesPlugin : Affiche valeurs € sur graphique barres
  • categoryPercentagePlugin : Affiche % sur graphique catégories

Optimisations :

  • clearRect() sur canvas si données vides (évite affichage vide)
  • Limitation Top 20 produits et Top 5 tendances
  • Truncation noms produits > 30 caractères
  • Animation pulse sur badges de notification

========================================== SOURCES DE DONNÉES

IPC Channels utilisés :

  • list-saved-quotes : Liste tous les devis (tous statuts)
  • list-invoices : Liste toutes les factures (tous statuts)

LocalStorage :

  • materialSearchHistory : Historique des recherches de matériaux

Filtres appliqués :

  • Devis : Seuls "accepté" et "facturé" comptent dans les ventes
  • Factures : Seules "payée" comptent dans le CA
  • Recherches : Filtrées par jour (aujourd'hui) ou semaine (7 derniers jours)

========================================== SYNCHRONISATION AVEC SCOPE GLOBAL

Variables synchronisées :

  • window.materialSearchHistory ↔ this.materialSearchHistory
  • window.adminDashboardCharts ↔ this.adminDashboardCharts

Fonctions exposées (compatibilité legacy) :

  • trackMaterialSearch, loadVendeurDashboard, loadAdminDashboard
  • updateVendeurPendingQuotes, updateVendeurQuotesWeek, updateVendeurSearchesToday
  • updateVendeurTopMaterials, updateAdminKPIs, updateAdminCharts
  • createQuotesStatusChart, createRevenueChart, createTopProductsChart
  • createCategoriesChart, analyzeProductData, displayTrendingProducts
  • generateRecommendation

========================================== INTÉGRATIONS

  • MaterialsModule : Tracking des recherches via trackMaterialSearch()
  • QuotesModule : Données pour KPIs et analytics
  • InvoicesModule : Données pour CA et analytics
  • Chart.js : Bibliothèque de graphiques (v3+)
  • Bootstrap 5.3.3 : Badges, animations, mise en page
