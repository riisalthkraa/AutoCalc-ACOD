UndoRedoManager

UndoRedoManager()

UNDO/REDO MANAGER

Gestionnaire universel d'annulation/rétablissement des actions. Inspiré de Linear, Notion, Figma pour offrir une expérience fluide.

Fonctionnalités :

  • Historique des 50 dernières actions
  • Ctrl+Z / Ctrl+Y (Undo/Redo)
  • Persistence localStorage (optionnel)
  • Groupement d'actions (batch)
  • Toast notifications avec bouton "Rétablir"
  • Support des actions asynchrones

Architecture :

  • undoStack: Actions déjà effectuées (can undo)
  • redoStack: Actions annulées (can redo)
  • Chaque action a: type, undo(), redo(), description

Examples
// Enregistrer une action
UndoRedoManager.record({
  type: 'MATERIAL_DELETE',
  description: 'Suppression matériau "Ciment"',
  undo: async () => { await restoreMaterial(material); },
  redo: async () => { await deleteMaterial(material.id); }
});
// Annuler la dernière action
await UndoRedoManager.undo();
// Rétablir
await UndoRedoManager.redo();
