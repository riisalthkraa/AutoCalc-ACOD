======================================== MODAL HELPER - Utilitaire de gestion des modals Bootstrap

Classe utilitaire pour gérer les modals Bootstrap 5.3.3 de manière centralisée. Réduit la duplication de code à travers tous les modules de l'application.

Architecture :

  • Méthodes statiques : Pas besoin d'instanciation, utilisation directe
  • Gestion d'erreurs : Logs clairs si modal introuvable
  • API simple : show(), hide(), getInstance(), toggle()
  • Délai de transition : Support pour éviter conflits Bootstrap

========================================== PROBLÈME RÉSOLU

Avant ModalHelper :

  • Code dupliqué dans 8+ modules
  • Gestion d'erreurs incohérente
  • Difficile à maintenir

Après ModalHelper :

  • Code centralisé
  • Gestion d'erreurs uniforme
  • ~100 lignes de code économisées

========================================== UTILISATION
