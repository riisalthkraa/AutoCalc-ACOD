======================================== LOADING STATE MANAGER

Gestionnaire centralisé des états de chargement pour l'application. Fournit skeleton screens, spinners, et progress bars.

Fonctionnalités:

  • Skeleton screens pour tables, cartes, listes
  • Spinners sur boutons
  • Progress bars pour opérations longues
  • États d'erreur avec retry
Version:
  • 1.0.0
Author:
  • AutoCalc OptiDevis - Phase 1 UX Improvements
