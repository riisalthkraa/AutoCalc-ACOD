# async init() → {Promise.<void>}

Initialise le module d'authentification

Workflow :

  1. Récupère les éléments DOM (switch, modal, input)
  2. Initialise le modal Bootstrap
  3. Démarre en mode Vendeur/Conseil (sécurité)
  4. Attache les event listeners
  5. Synchronise avec le StateManager
  6. Expose les fonctions globales (compatibilité)
  7. Écoute l'événement app:initialized

# async verifyPassword(password) → {Promise.<boolean>}

Vérifie le mot de passe auprès du backend (main process) Utilise bcrypt pour le hashing côté serveur

Workflow :

  1. Appel IPC 'verify-password' avec le mot de passe en clair
  2. Backend hash le mot de passe et compare avec la base
  3. Si valide → génération d'un token de session sécurisé
  4. Stockage du token dans localStorage et StateManager
Parameters:
Name Type Description
password string

Mot de passe à vérifier (en clair)
